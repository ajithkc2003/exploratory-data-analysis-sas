# Exploratory Data Analysis Using SAS

## Objective

This project focuses on performing Exploratory Data Analysis (EDA) using SAS to understand data characteristics, identify patterns, detect anomalies, and prepare data for further statistical modeling. The analysis involved data cleaning, descriptive statistics, correlation analysis, variable transformations, and visualization techniques to extract meaningful insights from the dataset.

---

## Skills Learned

- Exploratory Data Analysis (EDA)
- Data Cleaning and Preprocessing
- Descriptive Statistics
- Correlation Analysis
- Data Transformation Techniques
- Missing Value Analysis
- Outlier Detection
- Statistical Visualization
- Hypothesis Exploration
- Data Interpretation and Reporting
- SAS Programming
- Data-Driven Decision Making

---

## Tools Used

- **SAS Studio** for statistical analysis and data management
- **PROC MEANS** for descriptive statistics
- **PROC FREQ** for frequency analysis
- **PROC CORR** for correlation analysis
- **PROC UNIVARIATE** for distribution analysis
- **PROC SGPLOT** for data visualization
- **PROC SQL** for data manipulation and querying

---

## Project Workflow

### 1. Data Import and Preparation

The dataset was imported into SAS and prepared for analysis through:

- Data validation
- Missing value identification
- Data cleaning
- Variable formatting
- Data quality assessment

---

### 2. Descriptive Statistical Analysis

Summary statistics were generated to understand the dataset.

#### Measures Analyzed

- Mean
- Median
- Mode
- Standard Deviation
- Variance
- Minimum and Maximum Values
- Quartiles and Percentiles

These statistics provided an overview of central tendency and variability within the data.

---

### 3. Data Visualization

Several graphical techniques were used to understand data distributions and patterns.

#### Visualizations Created

- Histograms
- Box Plots
- Bar Charts
- Scatter Plots
- Frequency Charts
- Density Plots

These visualizations helped identify trends, skewness, and potential outliers.

---

### 4. Correlation Analysis

Relationships between variables were examined using correlation techniques.

#### Methods Used

- Pearson Correlation
- Correlation Matrix Analysis

#### Key Objectives

- Identify strongly related variables
- Detect multicollinearity
- Understand variable dependencies
- Support feature selection

---

### 5. Data Transformation

Several transformation techniques were applied to improve data quality and statistical assumptions.

#### Transformations Performed

- Log Transformation
- Standardization
- Normalization
- Variable Scaling

These transformations improved data distribution and model readiness.

---

### 6. Outlier Detection

Potential outliers were identified through:

- Box Plot Analysis
- Z-Score Methods
- Distribution Analysis

Outlier investigation helped ensure data reliability and improve analytical accuracy.

---

### 7. Insight Generation

The final phase focused on interpreting statistical results and generating actionable insights from the dataset.

#### Areas of Analysis

- Variable Relationships
- Distribution Patterns
- Data Quality Assessment
- Statistical Trends
- Business/Research Insights

---

## Key Findings

- Identified important patterns and trends within the dataset.
- Detected significant correlations among variables.
- Improved data quality through cleaning and transformation.
- Recognized outliers and unusual observations.
- Generated meaningful visual insights for decision-making.
- Established a strong foundation for predictive modeling and advanced analytics.
